# About
This project was to create a basic spotify-like web service using youtube.

# Project Requirements: 
- Python3
  - Installed through repositories
- Python Flask
  - Best installed with pip3
- youtube-dl
  - Best installed with pip3
- ffmpeg
  - Installed through repositories
- ffmpeg-normalize
  - Best installed with pip3
